Gantt Planet (甘特星球) Privacy Policy / 隱私權政策

Last Updated / 最後更新日期: 2026-02-03

🇺🇸 English

1. Overview

Gantt Planet (甘特星球) (hereinafter referred to as "this App") is a life habit tracking and goal management application developed by ChengChe Lee. We adhere to a strict privacy-first policy.

In short: We do NOT collect, store, or transmit any of your personal data to external servers.

2. Data Collection and Usage

Data We Do NOT Collect

This App does not collect:

  • ❌ Personally Identifiable Information (PII) such as name, email, or phone number
  • ❌ Location history or GPS coordinates (only used momentarily for weather)
  • ❌ Device identifiers or tracking IDs
  • ❌ Advertising data
  • ❌ Usage analytics or behavioral data

Locally Stored Data

The following data is stored strictly on your device and is never transmitted externally:

  • Habit/Goal Items: Titles, frequencies, colors, progress settings
  • Completion Records: Dates, timestamps
  • Diary Entries: Title, content, weather type
  • Tree Hole Conversations: Chat history and letter content
  • Art Reward Collection: Unlock progress
  • User Preferences: Language, display options
  • Diary Lock PIN (Optional): Securely stored in iOS Keychain

3. Location & Weather

  • Location Permission: "When In Use" only
  • Purpose: Fetch current weather via Apple WeatherKit
  • Accuracy: Kilometer-level only (not precise)
  • Storage: Only weather type is saved (not coordinates)
  • Optional: App functions fully without location permission

4. On-Device AI Features

This App provides completely offline AI chat capabilities:

  • Model Source: LLM models downloaded from HuggingFace Hub (requires separate download)
  • Local Processing: All AI inference runs 100% on-device using Metal GPU
  • Data Privacy: No chat data or diary content is sent to external servers
  • Supported Models: Qwen, Llama, Mistral, Phi, Gemma, SmolVLM

5. Network Access

Network access is limited to:

  • ✅ Weather data via Apple WeatherKit
  • ✅ LLM model downloads (user-initiated, from HuggingFace)
  • ✅ Art image downloads (from GitHub, user-earned rewards)
  • ✅ External links (App Store, privacy policy page)

Except for the above scenarios, this App does not establish any network connections.

6. Third-Party Services

Services We Do NOT Use

  • ❌ Google Analytics / Firebase
  • ❌ Facebook SDK
  • ❌ Any advertising networks
  • ❌ Crash reporting services
  • ❌ Cloud sync / backup services

Apple Services Used

  • ✅ WeatherKit (weather data)
  • ✅ LocalAuthentication (Face ID/Touch ID for diary lock)

7. Data Security

  • Local Storage: All user data stored in app sandbox
  • Optional Diary Encryption: Protect diary with PIN or biometrics
  • Secure Storage: PIN stored securely in iOS Keychain
  • No Cloud Backup: No cloud sync or backup available
  • Delete App = Delete Data: Uninstalling the app removes all stored data

8. Children's Privacy

This App is suitable for all ages. Since we don't collect any personal information, there are no special considerations for children's privacy. This App has no social features or user account system.

9. Your Rights

  • Access: View all your data anytime within the app
  • Deletion: Clear individual entries or all data
  • Control: Enable/disable location, notifications, diary lock

10. Contact Us

If you have any questions about this Privacy Policy, feel free to contact us:

🇹🇼 繁體中文

1. 概述

甘特星球（以下簡稱「本應用」）是一款由 ChengChe Lee 開發的生活習慣追蹤與目標管理應用程式。我們非常重視您的隱私。

簡單來說：本應用不會收集、儲存或傳輸您的任何個人資料到外部伺服器。

2. 資料收集與使用

我們不收集的資料

本應用不會收集以下資料：

  • ❌ 個人身份資訊（姓名、電子郵件、電話號碼等）
  • ❌ 位置歷史或 GPS 座標（僅暫時使用以取得天氣資訊）
  • ❌ 裝置識別碼或追蹤 ID
  • ❌ 廣告資料
  • ❌ 使用行為分析資料

本地儲存的資料

以下資料僅儲存在您的裝置上，不會傳輸至任何外部伺服器：

  • 習慣與目標項目： 標題、頻率、顏色、進度設定
  • 完成記錄： 日期、時間戳記
  • 日記項目： 標題、內容、天氣類型
  • 樹洞對話： 聊天記錄與信件內容
  • 藝術獎勵收藏： 解鎖進度
  • 使用者偏好設定： 語言、顯示選項
  • 日記鎖定 PIN（選用）： 安全儲存於 iOS 鑰匙圈

3. 位置與天氣

  • 位置權限： 僅在「使用應用程式期間」存取
  • 用途： 透過 Apple WeatherKit 取得即時天氣資訊
  • 精確度： 僅使用公里級別位置（非精確定位）
  • 儲存方式： 僅儲存天氣類型（不儲存座標）
  • 選用性： 即使不授權位置權限，應用程式仍可正常運作

4. 裝置端 AI 功能

本應用提供完全離線的 AI 聊天功能：

  • 模型來源： 從 HuggingFace Hub 下載大型語言模型（需另行下載）
  • 本地運算： 所有 AI 推論 100% 在裝置上使用 Metal GPU 執行
  • 資料隱私： 聊天內容與日記資料不會傳送至任何外部伺服器
  • 支援模型： Qwen、Llama、Mistral、Phi、Gemma、SmolVLM

5. 網路存取

本應用僅在以下情況存取網路：

  • ✅ 透過 Apple WeatherKit 取得天氣資料
  • ✅ 下載本地 AI 模型（使用者主動操作，來自 HuggingFace）
  • ✅ 下載藝術獎勵圖片（來自 GitHub，使用者獲得的獎勵）
  • ✅ 開啟外部連結（App Store、隱私權政策頁面）

除上述情況外，本應用不會主動建立任何網路連線。

6. 第三方服務

我們不使用的服務

  • ❌ Google Analytics / Firebase
  • ❌ Facebook SDK
  • ❌ 任何廣告網路
  • ❌ 當機回報服務
  • ❌ 雲端同步/備份服務

使用的 Apple 服務

  • ✅ WeatherKit（天氣資料）
  • ✅ LocalAuthentication（日記鎖定的 Face ID/Touch ID）

7. 資料安全

  • 本地端儲存： 所有使用者資料儲存於應用程式沙盒中
  • 選用日記加密： 可使用 PIN 或生物辨識保護日記
  • 安全儲存： PIN 安全儲存於 iOS 鑰匙圈
  • 無雲端備份： 不提供雲端同步或備份功能
  • 刪除應用 = 刪除資料： 解除安裝應用程式會移除所有儲存的資料

8. 兒童隱私

本應用適合所有年齡層使用。由於我們不收集任何個人資訊，因此沒有兒童隱私的特殊考量。本應用不包含社交功能或使用者帳戶系統。

9. 您的權利

  • 存取權： 您可以隨時在應用程式內查看您的所有資料
  • 刪除權： 可清除個別項目或所有資料
  • 控制權： 可自由啟用或停用位置、通知、日記鎖定等功能
