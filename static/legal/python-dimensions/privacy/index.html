Python Dimensions Privacy Policy / 隱私權政策

Last Updated / 最後更新日期: 2026-02-14

English

1. Overview

Python Dimensions (hereinafter referred to as "this App") is a Python programming language learning application developed by ChengChe Lee. It helps users prepare for PCEP, TQC+, and CPE certification exams through three question types: multiple choice, fill-in-the-blank, and code reordering. We adhere to a strict privacy-first policy.

In short: We do NOT collect, store, or transmit any of your personal data to external servers.

2. Data Collection and Usage

Data We Do NOT Collect

This App does not collect:

  • Personally Identifiable Information (PII) such as name, email, or phone number
  • Location data
  • Device Identifiers
  • Usage analytics or tracking data
  • Any form of tracking data

Locally Stored Data

The following data is stored strictly on your device and is never transmitted externally:

  • Practice Progress: Number of questions answered and accuracy for each question type
  • Error Records: Incorrect answers with spaced repetition review progress
  • Learning Analytics: Statistics across eight knowledge dimensions (for radar chart and heat map)
  • Conversation History: AI Assistant chat history
  • Preferences: Difficulty mode, exam type, and other user settings

3. Offline Features & AI

Core features work completely offline:

  • Practice Questions: All multiple choice, fill-in-the-blank, and code reordering questions are bundled within the app and require no internet connection.
  • AI Assistant: Uses on-device Large Language Models (requires separate download). All AI computation is performed locally. No user data is collected.
  • Python Playground: Runs code using an embedded Python interpreter on your device.

All AI processing is performed entirely offline; your data is never uploaded to any server.

4. Third-Party Services

This App does NOT use any third-party analytics or advertising frameworks, including:

  • Google Analytics
  • Facebook SDK
  • Firebase Analytics
  • Any advertising SDK
  • Crash reporting services

5. Network Access

Network access is restricted to the following limited scenarios:

  • Downloading AI Models (Optional): Only connects when you explicitly choose to download model resources. Models are stored locally after download and require no network for subsequent use.
  • External Links: Opens the browser when you tap "Privacy Policy" or "Developer" links.

Except for the above scenarios, this App does not establish any network connections.

6. Data Security

  • Local Storage: All data is stored only on your device
  • App Sandbox: Data is isolated within the app's sandbox container
  • App Groups: AI model files are shared locally via iOS App Groups, with no external transmission
  • iOS Security: Benefits from device encryption and iOS security mechanisms
  • Delete App = Delete Data: Uninstalling the app removes all stored data

7. Children's Privacy

This App is suitable for all ages. Since we don't collect any personal information, there are no special considerations for children's privacy.

8. Your Rights

  • Access: You can view your learning data anytime within the app
  • Deletion: You can reset all progress in Settings, or uninstall the app to completely delete all data
  • Control: All data is on your device, under your complete control

9. Policy Changes

If we update this Privacy Policy, we will revise the content on this page and update the "Last Updated" date.

10. Contact Us

If you have any questions about this Privacy Policy, feel free to contact us:

Email: qqder339@gmail.com

1. 概述

Python 點線面（以下簡稱「本應用」）是一款由 ChengChe Lee 開發的 Python 程式語言學習應用程式，透過選擇題、填空題與程式重組三種題型，幫助使用者準備 PCEP、TQC+ 及 CPE 認證考試。我們非常重視您的隱私。

簡單來說：本應用不會收集、儲存或傳輸您的任何個人資料到外部伺服器。

2. 資料收集與使用

我們不收集的資料

本應用不會收集以下資料：

  • 個人身份資訊（姓名、電子郵件、電話號碼等）
  • 位置資訊
  • 裝置識別碼 (Device ID)
  • 使用行為分析資料
  • 任何形式的追蹤資料 (Tracking Data)

本地儲存的資料

以下資料僅儲存在您的裝置上，不會傳輸至任何外部伺服器：

  • 練習進度： 各題型的作答數量與正確率
  • 錯題記錄： 答錯的題目與間隔重複複習進度
  • 學習分析： 八大知識維度的答題統計（用於雷達圖與熱力圖）
  • 對話記錄： AI 助教功能的對話歷史
  • 偏好設定： 難度模式、考試類型等使用者設定

3. 離線功能與 AI

本應用的核心功能皆可在完全離線的狀態下運作：

  • 練習題目： 所有選擇題、填空題、程式重組題均內建於應用中，不需網路連線。
  • AI 助教： 使用裝置端大型語言模型（需另行下載），所有 AI 運算都在本地完成，不收集任何使用者資料。
  • Python 遊樂場： 使用內嵌的 Python 直譯器在裝置上執行程式碼。

所有 AI 處理過程完全離線，您的資料不會上傳到任何伺服器。

4. 第三方服務

本應用 不使用 任何第三方分析或廣告服務，包括但不限於：

  • Google Analytics
  • Facebook SDK
  • Firebase Analytics
  • 任何廣告 SDK
  • 當機回報服務

5. 網路存取

本應用僅在以下極少數情況存取網路：

  • 下載本地 AI 模型（選用功能）： 僅在您主動選擇下載時進行連線。模型下載後儲存於裝置本地，後續使用不需網路。
  • 外部連結： 當您主動點擊「隱私權政策」或「開發者」連結時開啟瀏覽器。

除上述情況外，本應用不會主動建立任何網路連線。

6. 資料安全

  • 本地端儲存： 所有資料都儲存在您的裝置上
  • App 沙盒： 資料隔離在應用程式的沙盒容器中
  • App Groups： AI 模型檔案透過 iOS App Groups 於本地共享，不涉及外部傳輸
  • iOS 安全性： 受益於裝置加密和 iOS 安全機制
  • 刪除應用 = 刪除資料： 解除安裝應用程式會移除所有儲存的資料

7. 兒童隱私

本應用適合所有年齡層使用。由於我們不收集任何個人資訊，因此沒有兒童隱私的特殊考量。

8. 您的權利

  • 存取權： 您可以隨時在應用程式內查看您的學習資料
  • 刪除權： 您可以在設定中重置所有進度，或解除安裝應用程式以完全刪除所有資料
  • 控制權： 所有資料都在您的裝置上，由您完全掌控
