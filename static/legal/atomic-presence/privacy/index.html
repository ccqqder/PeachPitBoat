AtomicPresence Privacy Policy / 隱私權政策

Last Updated / 最後更新日期: 2026-02-14

English

1. Overview

AtomicPresence (原子受訪, hereinafter referred to as "this App") is an interview protection application developed by Chengche Lee. It uses cryptographic hash chains and digital signatures to provide interviewees with tamper-proof, self-verifiable evidence capabilities. We adhere to a strict privacy-first policy.

In short: We do NOT collect, store, or transmit any of your personal data to external servers. All recordings, videos, and signature data are stored exclusively on your device. No user data is collected.

2. Data Collection and Usage

Data We Do NOT Collect

This App does not collect:

  • Personally Identifiable Information (PII) such as name, email, or phone number
  • Location data
  • Device Identifiers
  • Usage analytics or tracking data

Locally Stored Data

The following data is stored strictly on your device and is never transmitted externally:

  • Audio Recordings (Level 2): M4A audio evidence files stored in the app's Documents directory.
  • Video Recordings (Level 3): MP4 video evidence files stored in the app's Documents directory.
  • Digital Signature Evidence: JSON files containing hash chains, timestamps, and Curve25519/Ed25519 signatures, stored in the app's Documents directory.
  • Signing Keys: Curve25519 signing keys securely stored in the iOS Keychain, existing only on your device.
  • Preferences: Theme selection, audio quality mode, and other settings stored locally in UserDefaults.

3. Device Permissions

This App requires the following device permissions, all used exclusively for local functionality:

  • Microphone: Used in Level 2 (Witness mode) to record audio evidence. Audio is stored only on your device.
  • Camera: Used in Level 3 (Sovereign mode) to record video evidence. Video is stored only on your device.
  • Photo Library: Optionally used to save video evidence to your Photo Library, only when you explicitly choose to do so.

All data captured via microphone and camera is never uploaded to any server.

4. Cryptography and Security

This App uses Apple's CryptoKit framework for all cryptographic operations:

  • Hash Chains: Generated in real-time on-device using SHA-256, with no network communication involved.
  • Digital Signatures: Uses Curve25519/Ed25519 algorithms. Keys are generated locally and stored in the iOS Keychain.
  • No Cloud Verification: All evidence verification is performed entirely on-device, requiring no server connection.

No user data is collected.

5. Third-Party Services

This App does NOT use any third-party analytics or advertising frameworks (No Google Analytics, No Facebook SDK, No Firebase, No Ads, No Crash Reporting).

6. Network Access

This App requires no network connection. All features — hash chain generation, QR code display, audio recording, video recording, digital signing, and evidence verification — work completely offline. The App does not initiate any network connections.

7. Data Control

  • All evidence files are directly accessible and manageable through the iOS Files app.
  • Deleting the app removes all local data, including signing keys stored in the Keychain.
  • No account creation or login required.

8. Contact Us

If you have any questions regarding this Privacy Policy, please contact us at:

繁體中文

1. 概述

AtomicPresence（原子受訪，以下簡稱「本應用」）是一款由 Chengche Lee 開發的受訪保護應用程式，透過密碼學雜湊鏈與數位簽章技術，為受訪者提供防偽造、防篡改的自我驗證能力。我們非常重視您的隱私。

簡單來說：本應用不會收集、儲存或傳輸您的任何個人資料到外部伺服器。所有錄音、錄影與簽章資料僅儲存在您的裝置上。不收集任何使用者資料。

2. 資料收集與使用

我們不收集的資料

本應用不會收集以下資料：

  • 個人身份資訊（姓名、電子郵件、電話號碼等）
  • 位置資訊
  • 裝置識別碼 (Device ID)
  • 使用行為分析資料
  • 任何形式的追蹤資料 (Tracking Data)

本地儲存的資料

以下資料僅儲存在您的裝置上，不會傳輸至任何外部伺服器：

  • 錄音檔案（Level 2）： M4A 格式的音訊證據檔案，儲存於裝置的 Documents 目錄。
  • 錄影檔案（Level 3）： MP4 格式的影片證據檔案，儲存於裝置的 Documents 目錄。
  • 數位簽章證據： 包含雜湊鏈、時間戳與 Curve25519/Ed25519 簽章的 JSON 檔案，儲存於裝置的 Documents 目錄。
  • 簽章金鑰： 使用 iOS Keychain 安全儲存的 Curve25519 簽章金鑰，僅存在於您的裝置上。
  • 偏好設定： 主題選擇、音訊品質模式等設定（儲存於裝置本地 UserDefaults）。

3. 裝置權限

本應用需要以下裝置權限，所有權限僅用於本地功能：

  • 麥克風： Level 2（見證模式）錄製音訊證據時使用。音訊僅儲存在您的裝置上。
  • 相機： Level 3（主權模式）錄製影片證據時使用。影片僅儲存在您的裝置上。
  • 照片圖庫： 選擇性將影片證據儲存至您的照片圖庫。僅在您主動操作時使用。

所有透過麥克風和相機擷取的資料都不會上傳至任何伺服器。

4. 密碼學與安全

本應用使用 Apple CryptoKit 框架進行所有密碼學運算：

  • 雜湊鏈： 使用 SHA-256 在裝置上即時產生，不涉及任何網路通訊。
  • 數位簽章： 使用 Curve25519/Ed25519 演算法，金鑰由裝置本地產生並儲存於 iOS Keychain。
  • 無雲端驗證： 所有證據驗證都在裝置本地完成，不需要連線至任何伺服器。

不收集任何使用者資料。

5. 第三方服務

本應用 不使用 任何第三方分析或廣告服務，包括但不限於：

  • Google Analytics
  • Facebook SDK
  • Firebase
  • 任何廣告 SDK
  • 任何崩潰報告服務

6. 網路存取

本應用 不需要網路連線。所有功能（雜湊鏈產生、QR 碼顯示、錄音、錄影、數位簽章、證據驗證）均完全離線運作。本應用不會主動建立任何網路連線。

7. 資料控制

  • 所有證據檔案透過 iOS Files 應用程式可直接存取與管理。
  • 刪除應用程式即刪除所有本地資料（包括 Keychain 中的簽章金鑰）。
  • 不需要建立帳號，不需要登入。
