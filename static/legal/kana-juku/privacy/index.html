Kana Juku (假名私塾) Privacy Policy / 隱私權政策

Last Updated / 最後更新日期: 2026-03-15

🇺🇸 English

1. Overview

Kana Juku (hereinafter referred to as "this App") is a Japanese Kana learning application developed by Chengche Lee. We adhere to a strict privacy-first policy.

In short: We do NOT collect, store, or transmit any of your personal data to external servers.

2. Data Collection and Usage

Data We Do NOT Collect

This App does not collect:

  • Personally Identifiable Information (PII) such as name, email, or phone number
  • Location data
  • Device Identifiers
  • Usage analytics or tracking data

Locally Stored Data

The following data is stored strictly on your device and is never transmitted externally:

  • Learning Progress: Stored locally to track your learning status.
  • User Settings: Stored locally to save your preferences.
  • Handwriting Input: Processed in real-time memory and discarded immediately; no files are saved.
  • Widget Data: Uses App Groups shared container to display Kana on widgets.

3. Offline Features & AI

Core features work completely offline:

  • Handwriting Recognition: Uses on-device Core ML models; all processing is local.
  • Text-to-Speech: Uses pre-downloaded audio assets.
  • AI Assistance: Uses local Large Language Models (LLM). Inference is performed on-device without data upload.

4. Third-Party Services

This App does NOT use any third-party analytics or advertising frameworks (No Google Analytics, No Facebook SDK, No Ads).

5. Network Access

Network access is restricted to:

  • Downloading AI Models (Optional): Only connects when you explicitly choose to download the local model resources.
  • External Links:
    • Opens the browser when you tap "Rate on App Store" or "Privacy Policy".
    • Opens the browser when you tap the "Search Web" button after using the translation/recognition feature to look up detailed explanations of words (e.g., linking to online dictionaries or search engines).

Other than the above, the App does not initiate network connections.

If you have any questions regarding this Privacy Policy, please contact us at:

🇹🇼 繁體中文

1. 概述

假名私塾（以下簡稱「本應用」）是一款由 Chengche Lee 開發的日文假名學習應用程式。我們非常重視您的隱私。

簡單來說：本應用不會收集、儲存或傳輸您的任何個人資料到外部伺服器。

2. 資料收集與使用

我們不收集的資料

本應用不會收集以下資料：

  • 個人身份資訊（姓名、電子郵件、電話號碼等）
  • 位置資訊
  • 裝置識別碼 (Device ID)
  • 使用行為分析資料
  • 任何形式的追蹤資料 (Tracking Data)

本地儲存的資料

以下資料僅儲存在您的裝置上，不會傳輸至任何外部伺服器：

  • 學習進度： 記錄您的假名學習狀態（儲存於裝置本地）。
  • 使用者設定： 儲存您的偏好設定（儲存於裝置本地）。
  • 手寫輸入： 即時辨識後立即丟棄，僅存於記憶體，不儲存檔案。
  • 小工具資料： 透過 App Groups 共享容器顯示假名於桌面小工具。

3. 離線功能與 AI

本應用的核心功能皆可在完全離線的狀態下運作：

  • 手寫辨識： 使用裝置內建的 Core ML 模型，所有運算都在本地完成。
  • 語音合成： 使用預先下載的音訊檔案。
  • AI 翻譯輔助： 使用本地大型語言模型（需另行下載），運算過程完全離線，不需上傳資料。

4. 第三方服務

本應用 不使用 任何第三方分析或廣告服務，包括但不限於：

  • ❌ Google Analytics
  • ❌ Facebook SDK
  • ❌ 任何廣告 SDK

5. 網路存取

本應用僅在以下極少數情況存取網路：

  • 下載本地 AI 模型（選用功能）： 僅在您主動選擇下載時進行連線。
  • 外部連結：
    • 當您主動點擊「App Store 評分」或「隱私權政策」時開啟瀏覽器。
    • 當您使用翻譯/辨識功能後，主動點擊「查詢網頁」按鈕以在瀏覽器中搜尋該詞彙的詳細解釋（如連結至線上字典或搜尋引擎）。

除上述情況外，本應用不會主動建立任何網路連線。
