## 1. Обзор

Atomic Presence, разработанное QQder339, — это anti-deepfake инструмент, который использует криптографические Hash chains, цифровые подписи и audio watermarking, чтобы пользователи могли самостоятельно проверять целостность своих записей.

**Кратко: мы НЕ собираем, НЕ храним и НЕ передаём ваши персональные данные на внешние серверы. Все криптографические операции и проверки выполняются на устройстве.**

## 2. Какие данные мы НЕ собираем

Это приложение не собирает:

- Персональные идентифицирующие данные (имя, email, номер телефона)
- Данные о местоположении
- Идентификаторы устройства
- Аналитику использования или данные трекинга

## 3. Данные, хранящиеся локально

Следующие данные хранятся строго на вашем устройстве и никогда не передаются наружу:

- **Аудио/видео файлы**: весь записанный контент хранится в локальном хранилище устройства
- **Записи Hash chain**: последовательности SHA-256 hash и соответствующие данные проверки
- **Цифровые подписи**: данные подписи, созданные алгоритмом Curve25519 на устройстве
- **Отчёты проверки**: отчёты целостности и записи metadata
- **Анонимизированный идентификатор устройства**: каждый `.evidence.json` содержит 16-символьный hex prefix от `SHA-256(identifierForVendor)`, который используется только для сопоставления записей с одного устройства при проверке. Этот идентификатор существует только внутри файлов доказательств на вашем устройстве, никогда не передаётся ни на какой сервер и не может быть обращён обратно к исходной информации устройства

## 4. Криптографические функции (полностью offline)

Все ключевые функции выполняются на устройстве без сетевого подключения:

- **Генерация Hash chain**: последовательности SHA-256 hash в реальном времени; все вычисления выполняются локально
- **Цифровая подпись**: используется алгоритм Curve25519 для подписи записей на устройстве
- **Audio watermarking**: встраивание FSK-сигналов в записи; вся обработка сигнала выполняется на устройстве
- **Проверка**: вычисление проверки целостности выполняется локально

## 5. Важное примечание

Контент, обрабатываемый приложением (аудио, видео), может содержать чувствительную информацию. Вся обработка происходит на вашем устройстве, и **мы не можем и никогда не будем иметь доступ к вашим записям**.
