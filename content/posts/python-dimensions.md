## Python 點線面

> 你的隨身 Python 離線家教

Python 點線面以「點、線、面」三個維度構建 Python 學習體驗，從選擇題到程式碼排列，再到 AI 互動教學，系統性地掌握 Python。

### 核心特色

- **三維訓練模式** — 選擇題（MCQ）、填空題（Cloze）、程式碼排列（Parsons）
- **裝置端 AI 家教** — 搭配 RAG 技術，提供精準的 Python 知識問答
- **內建 Python 3.13 執行環境** — 直接在 App 內撰寫並執行 Python 程式碼
- **300+ 題庫** — 涵蓋 PCEP、TQC+、CPE 等認證考試
- **雷達圖分析** — 視覺化呈現你的各項能力分布
- **14 個可下載 AI 模型** — 依需求選擇最適合的模型
- **日式文具美學** — 以日本文具為靈感的介面設計
- **不收集任何使用者資料** — 所有運算在裝置端完成
