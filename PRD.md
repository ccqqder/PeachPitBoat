Hugo，因為它的主題生態系很多針對「開發者作品集」的優秀模版。託管 (Hosting)： GitHub Pages。理由：免費、穩定、版本控制即發布、符合 Hacker 精神。網域 (Domain)： 建議購買 .io (tech) 或 .me (個人品牌) 網域，設定 CNAME 指向 GitHub Pages。評論系統： Giscus (基於 GitHub Discussions) 或
